# Proyecto web del Dr. Felipe Hernández Serna

## Instrucciones para Claude Code

Antes de realizar cualquier cambio, lee todos los archivos de la carpeta `docs`.

## Reglas principales

- Este proyecto es independiente de NURA System.
- No inventar información médica, profesional o personal.
- No inventar servicios, estadísticas ni testimonios.
- No realizar cambios estructurales sin presentar primero un plan.
- Trabajar una tarea pequeña por vez.
- Priorizar simplicidad, accesibilidad, velocidad, SEO y seguridad.
- El sitio debe funcionar correctamente en teléfonos móviles.
- Mantener una imagen médica, humana, elegante y profesional.
- Documentar decisiones y cambios importantes.

## Flujo obligatorio

1. Analizar la tarea.
2. Revisar los documentos y archivos relacionados.
3. Presentar una propuesta.
4. Justificar la decisión técnica.
5. Esperar aprobación cuando sea un cambio importante.
6. Implementar.
7. Probar.
8. Informar qué archivos fueron modificados.
