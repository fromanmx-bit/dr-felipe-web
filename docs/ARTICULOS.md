# Artículos y recursos

## Regla general

Los artículos serán educativos, claros y prudentes.

No deberán:

- diagnosticar;
- recomendar medicamentos;
- dar dosis;
- prometer resultados;
- sustituir una consulta;
- generar alarma.

Cada artículo debe incluir el aviso:

“Este contenido es informativo y no sustituye una valoración médica.”
