# Identidad visual

## Logotipo

El símbolo principal es un roble formado por huellas de bebés.

## Significado

El concepto representa:

- vida;
- crecimiento;
- fortaleza;
- protección;
- acompañamiento;
- desarrollo;
- la historia particular de cada recién nacido.

## Lema

“Cada nueva huella, una historia que contar.”

## Colores base

- azul profundo;
- turquesa;
- blanco;
- gris neutro;
- dorado suave únicamente como acento.

Los códigos exactos de color todavía deben extraerse del logotipo
o definirse durante el sistema visual.

## Dirección estética

La identidad debe ser pediátrica y neonatal, pero no infantil.

El árbol y las huellas deben utilizarse con elegancia y moderación.

Evitar:

- saturar la interfaz con huellas;
- caricaturas innecesarias;
- colores excesivamente infantiles;
- imágenes clínicas fuertes;
- fotografías genéricas que reduzcan la confianza.

## Concepto de comunicación

La marca debe combinar:

- ciencia;
- experiencia;
- tranquilidad;
- cercanía;
- cuidado;
- crecimiento.

## Frase profesional disponible

“La mejor medicina se practica con ciencia y con corazón.”
