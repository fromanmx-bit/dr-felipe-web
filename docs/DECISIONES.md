# Registro de decisiones

## Decisión 001

La nueva página se desarrollará desde cero y no mediante correcciones
menores a la web anterior.

Motivo:
La página anterior contiene información provisional, imágenes genéricas
y una identidad profesional poco clara.

## Decisión 002

Se conservará el concepto del roble formado por huellas de bebés.

Motivo:
El símbolo tiene relación directa con neonatología, crecimiento,
protección y acompañamiento.

## Decisión 003

No se publicará ningún servicio, testimonio, estadística o dato profesional
que no haya sido confirmado.
