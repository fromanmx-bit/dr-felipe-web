# Contexto del proyecto

## Proyecto

Reconstrucción completa del sitio web profesional del
Dr. Felipe Hernández Serna.

## Independencia

Este proyecto no pertenece a NURA System y no debe mezclarse con él.

## Objetivo

Crear un sitio web moderno, elegante, rápido y profesional que:

- explique claramente quién es el médico;
- comunique sus especialidades;
- facilite la solicitud de consultas;
- transmita confianza y trato humano;
- funcione correctamente en dispositivos móviles;
- tenga una estructura adecuada para SEO;
- evite información falsa o no confirmada.

## Público principal

- padres de recién nacidos;
- padres de bebés y niños;
- familias que buscan un pediatra;
- familias que necesitan atención neonatal;
- pacientes que requieren seguimiento después de una hospitalización.

## Personalidad del sitio

Debe sentirse:

- médico;
- humano;
- tranquilo;
- especializado;
- confiable;
- moderno;
- sobrio.

No debe parecer:

- una guardería;
- una plantilla médica genérica;
- publicidad exagerada;
- una página hospitalaria fría;
- una interfaz recargada.
